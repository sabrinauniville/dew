# projeto - Angular 21 SPA de exemplo

Este projeto é uma aplicação Angular de exemplo que demonstra o uso de componentes standalone, rotas e consumo de APIs.
Ele foi criado para ilustrar como montar uma Single Page Application moderna com Angular 21.

## Tecnologias utilizadas e características

- Angular 21
- TypeScript
- HTML
- CSS
- RxJS
- Angular Router
- Standalone components

## Setup

Requisitos:
- Node.js 20+
- npm 10+

Como rodar:

```bash
cd js/22/projeto
npm install
npm start
```

Build de produção:

```bash
npm run build
```
