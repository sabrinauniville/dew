# Conteúdo do curso

Este documento será usado para armazenar o conteúdo completo de cada aula do aprendaweb.

## Instruções
- Produzir todo o conteúdo em PT-BR.
- O conteúdo final deve incluir títulos, descrições, tópicos, vídeos, exercícios e datas de liberação.
- Cada aula deve ter:
  - data de liberação;
  - data máxima de execução dos exercícios;
  - descrição clara do objetivo;
  - lista de tópicos;
  - exercícios práticos com opções de IA.
- Os exercícios devem ter formatos como:
  - quiz de seleção única;
  - quiz de múltiplas respostas;
  - escreva um código;
  - complete um código;
  - respostas em texto.
- Deve ser possível gerar esses conteúdos com IA a partir do título e da descrição do módulo e da aula.
